# The Holy Family’s Journey to Egypt

## Overview

The Holy Family’s Journey to Egypt is a front-end web project that documents and presents the historical and spiritual path taken by the Holy Family across Egypt. The project focuses on storytelling, visual presentation, and accessibility for Arabic content.

## Features

- Landing page introducing the journey
- Regions section with interactive cards
- Dedicated pages for each region
- Image carousels and galleries
- Responsive and mobile-friendly design
- Arabic (RTL) content support

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla)
- Font Awesome
- Google Fonts (Cairo)

## Project Structure

- `index.html` – Main landing page
- `*.html` – Region-specific pages
- `styles/` – CSS files
- `scripts.js/` – JavaScript files
- `images/` – Images and media assets

## Purpose

This project demonstrates front-end development skills, UI/UX design, and the ability to deliver a culturally focused web experience using modern web technologies.
